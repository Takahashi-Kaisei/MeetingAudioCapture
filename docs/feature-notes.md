# 機能追加ノート

MeetingAudioCapture に追加済みの機能と、今後の保守で気をつける制約をまとめます。実機受け入れテストは完了済みのため、このノートでは手順化せず、実装済みの挙動と運用上の注意に絞ります。

## 録音モード

- `オンライン会議`: システム音声と選択マイクを1つの音声ファイルへ保存します。
- `対面`: 選択マイクのみを保存します。
- システム音声は ScreenCaptureKit で取得します。v1では BlackHole、Loopback、Soundflower は不要です。
- 内蔵スピーカー利用時は相手音声がマイクへ回り込む可能性があります。ASR品質を重視する場合はイヤホンまたはヘッドセットを推奨します。

## 録音操作

- メニューバーには録音中の経過時間を `REC HH:MM:SS` 形式で表示します。
- 一時停止/再開に対応しています。一時停止中に届いた音声は破棄し、再開時はミキサーのタイムラインを作り直すため、停止中の長い無音は書き込まれません。
- 録音中の失敗は `ERR` 表示、`再試行`、`エラーをクリア` で復旧できる状態に遷移します。

## ファイル出力

- 保存先はメニューから変更でき、`UserDefaults` に保存します。保存先が使えない場合は Downloads に戻します。
- ファイル名にはアプリ名、日時、録音モード、任意タイトル、`partNNN` の分割番号を含めます。
- デフォルトの分割間隔は30分です。
- 出力形式:
  - M4A: AAC。デフォルト形式です。
  - WAV: PCM。ロスレスなローカル処理向けです。
  - MP3: 一時M4Aを作成し、`ffmpeg -i input.m4a -ar 44100 -ab 128k output.mp3` 相当で変換します。

## 分割ファイル結合

- `停止後に分割ファイルを結合` は任意設定で、`UserDefaults` に保存します。
- 結合は2つ以上の分割ファイルがある場合のみ実行します。
- 結合成功時は `_merged` ファイルだけを残します。
- 結合失敗時は復旧用に元の `partNNN` ファイルをすべて残します。
- M4A は AVFoundation の composition/export、WAV は PCM 連結、MP3 は `ffmpeg` concat で結合します。

## 保守メモ

- MP3出力とMP3結合には、Homebrewまたはシステム上の既知パスに `ffmpeg` が必要です。
- 自動テストは、ファイル名、経過時間、設定保存、ミキサー、出力形式、分割保存、結合、エラー文言などの純粋ロジックを中心にします。
- 実際の会議アプリ録音は macOS 権限、音声デバイス、会議アプリ状態に依存します。自動完了条件にはライブ録音を含めないでください。
